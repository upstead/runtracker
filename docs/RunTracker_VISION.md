# RunTracker - Product Vision

## Product Name

RunTracker

Optional publishing names:

- RunTracker by Upstead
- Upstead RunTracker

---

# Vision

RunTracker is a lightweight, offline-first Android application designed for people who simply want to track their running progress without creating accounts, sharing data, or connecting wearables.

---

# Core Philosophy

- No accounts
- No subscriptions
- No advertisements
- No social features
- No cloud dependency
- No unnecessary metrics

Track running progress simply and effectively.

---

# Target Audience

Primary audience:

- Casual runners
- Returning runners
- Privacy-conscious users

---

# Product Positioning

RunTracker is not intended to compete with:

- Strava
- Garmin Connect
- Nike Run Club
- Google Fit

Instead it focuses on:

- Local ownership of data
- Simplicity
- Fast logging
- Lightweight experience

---

# Privacy Principles

All user data remains on the device.

- No analytics
- No telemetry
- No tracking
- No account creation
